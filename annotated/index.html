Tableling

Table plugin based on Backbone Marionette.

Check out the demo!

v0.0.4: No test suite written yet. Consider this very alpha.

Requirements

Tableling currently works with the following versions of these libraries:

  • Underscore v1.3.3
  • Backbone v0.9.2
  • Backbone.Marionette v0.10.2

No other versions have been tested. It may or may not work using different versions.

Installation

Core Plugin

  • Development: tableling.js
  • Production: tableling.min.js

Bootstrap Views

  • Development: tableling-bootstrap.js
  • Production: tableling-bootstrap.min.js

Ruby on Rails

The tableling-rails gem provides the latest tableling release as well as backend code to easily build responses in the expected format.
